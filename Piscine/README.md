# 42 Piscine - BSQ
A 42 (school) project in which, given a map filled with obstacles you have to find the biggest square that fits within the empty tiles of the map.

## How to run
```shell
./bsq <map.txt>
```

## Example of a map:

..o..oo..o........

...o..o....o.o..o.
..oo...o.....o..oo
oo......o....o...o
...o....oo..oo....

o......o.o...o.o.o
........oo...o....

- '.' - empty tile
- 'o' - obstacle
- 'x' - square tile
